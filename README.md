# 上海红祺腾达信息技术有限公司静态官网

本项目为纯静态网站，可直接部署到 GitHub Pages、Cloudflare Pages 或 Vercel。

## 页面

- `index.html`：公司首页
- `catalog.html`：服务与产品查询
- `assets/data/catalog.json`：查询数据
- `assets/images/`：公司 Logo 与企业微信素材

## GitHub Pages

1. 将全部文件上传到公开仓库根目录。
2. 进入 `Settings → Pages`。
3. 选择 `Deploy from a branch`。
4. 选择 `main` 和 `/ (root)`。
5. 保存后等待生成访问地址。
